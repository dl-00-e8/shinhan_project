**1단계: 프로젝트 디렉토리 생성 및 이동**
```bash
mkdir -p project
cd project/server
```

**2단계: 가상환경 설정 (권장)**
```bash
# 가상환경 생성
python3 -m venv venv

# 가상환경 활성화
source venv/bin/activate  # macOS/Linux
# 또는
venv\Scripts\activate     # Windows
```

**3단계: requirements.txt 생성 방법**

```bash
# 설치된 패키지 목록으로 requirements.txt 생성
pip freeze > requirements.txt
```

**4단계: 패키지 설치**
```bash
# requirements.txt로 패키지 설치
pip install -r requirements.txt
```

**5단계: 설치 확인**
```bash
# 설치된 패키지 확인
pip list

# 특정 패키지 확인
python -c "import flask; print('Flask 설치 완료')"
python -c "import librosa; print('Librosa 설치 완료')"
```

## 주요 패키지 설명

- **Flask**: 웹 프레임워크 핵심
- **Flask-SQLAlchemy**: 데이터베이스 ORM
- **Flask-JWT-Extended**: JWT 토큰 인증
- **librosa**: 음성 신호 처리
- **scikit-learn**: 머신러닝 (성문 인식)
- **psycopg2-binary**: PostgreSQL 연결
- **redis**: 캐시 및 세션 관리
